# Robot Game

Jogo de movimentos de um robô em um espaço delimitado.

## Instalar Dependências

A partir do diretório da aplicação:

    gem install bundler --pre
    bundle

## Executar os Testes

A partir do diretório da aplicação:

    rspec

## Jogar

A partir do diretório da aplicação:

    ruby bin/robot.rb spec/fixtures/input_01.txt
